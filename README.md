# 🩺 Disease Prediction using Machine Learning

## 📌 Overview
This project demonstrates how to build a **disease prediction model** using machine learning.  
It leverages medical data (e.g., age, blood pressure, cholesterol, glucose levels) to predict whether a patient is likely to have a disease.  

The model uses **Random Forest Classifier** for robust predictions, with preprocessing steps like feature scaling and train-test splitting.

---

## ⚙️ Features
- End-to-end pipeline: data loading → preprocessing → training → evaluation → prediction
- Uses **Random Forest Classifier** (can be swapped with Logistic Regression, XGBoost, etc.)
- Evaluation metrics: Confusion Matrix, Classification Report, ROC-AUC
- Example prediction for a new patient
